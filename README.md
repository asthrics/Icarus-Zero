# Icarus-Zero

**ICARUS-ZERO** is a C# reactor simulation project powered by the **IZLS (Icarus Zero Logic System)**. It is a learning project focused on building structured systems, modular design, and simulation logic.

The goal of this project is to simulate a reactor using connected logic components while learning how to design software, organize code, and work with larger system structures.

The **IZLS** is the main logic system. It connects and controls the different parts of the reactor and makes sure everything works together.

This project is still in development and made by a beginner programmer. The code will change a lot over time as new features are added and old parts are improved or replaced.

If something breaks, it’s just the code becoming self-aware or something.
